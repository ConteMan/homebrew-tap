# ConteMan's Tap

Homebrew tap 与 Scoop bucket 合一的分发仓：`Casks/` 由 [GoReleaser](https://goreleaser.com) 在各项目发版时自动更新，`bucket/` 同理。请勿手动编辑。

## macOS (Homebrew)

```sh
brew install conteman/tap/repolens
```

## Windows (Scoop)

```powershell
scoop bucket add conteman https://github.com/ConteMan/homebrew-tap
scoop install repolens
```
